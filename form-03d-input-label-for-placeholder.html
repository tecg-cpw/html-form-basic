3d. Champ de saisie texte et attribut placeholder

1°) Sans placeholder (avec value)

C’est l’attribut value qui contient la valeur par défaut affichée dans le champ. Cette valeur ne se désactive pas quand on commence à remplir le champ (il faut la sélectionner pour l’effacer).

2°) Avec placeholder

C’est l’attribut placeholder qui contient la valeur par défaut affichée dans le champ. Cette valeur se désactive automatiquement quand on commence à remplir le champ.
